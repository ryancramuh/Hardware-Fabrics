# FreeRTOS Microblaze with AXI GPIO + UART 

## Steps 
 1. Create Block Design 
 2. Instantiate Microblaze and AXI GPIO/UART
 3. Validate Block Design
 4. Generate Output Products 
 5. Create HDL Wrapper 
 6. Generate Bitstream
 7. Export Hardware (.xsa) 
 8. Create a Platform Component in Vitis based on your hardware (.xsa)
 9. Create your Application Component in Vitis based on your Platform 
 10. Build and Run
